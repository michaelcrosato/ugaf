strictly-easier-than-slip button that "deleted the climax." This cycle made it a **hands-on "baggage gets
handled" pass** (`src/modules/spine/index.ts`, tag `debt_walkout`):

  already makes the cost legible in the epilogue.

## The honest limitation (the synth's #2, behaviour-anchored, HIGH) — the keystone advanced, the frontier moved
The debt is no longer the strictly-easier button — but the synth's sharpest finding is that **the cost is
soft for the *optimal* player**, because by the gate the iron is already dead weight. `systems-minmaxer`:
*"the gate only punishes iron, so `drop crowbar → hide → walk out` is a free, dominant, zero-cost
solution — I'd already used the crowbar, so shedding it cost me nothing."* `debt-route-critic`: *"the
material cost is illusory, because the thing it takes is dead weight."* The complaint has **migrated from
the debt route to the SLIP route**: the remaining free exit is `drop iron → hide → slip`, which night22 did
not (and was not scoped to) touch. The keystone's stated AC is met; the broader 0024 frontier — *"the
optimal path refunds every cost"* — persists and is now precisely localized to the gate as a whole.
